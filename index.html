Cheat Sheet

HTML Forms

elements

Tag Description Attributes
<form> a HTML element representing a section containing interactive controls for submitting information action (url where the form data gets sent), method(the http request used to send the form data)
<label> a caption for an item - clicking the text will activate the input must have "for" attribute which links it to the input element's "id"
<input> creates interactive controls so that the user can submit info see table below
<fieldset> a group of controls +/ labels in a form name
<legend> a caption for a fieldset none needed
<textarea> multi-line plain text submission area rows, cols, id, name
<select> creates a dropdown list. must be used with "option" id, name
<option> options in a dropdown menu value

input element - attributes

Attribute Description
<type> determines what kind of data the input element accepts
<name> necessary for the form data to be sent - paired with value when the form is submitted
<value> when users input data it becomes the value - paired with name when the form is submitted
<id> necessary to link an input with a label
<min/maxlength> sets a min or max length for text
<min/max> sets a min or max for numbers
<required> will throw an error if the input is not filled in
<pattern> sets the characters allowed in a field e.g [a-zA-Z0-9]+
